# 自适应PID控制器（以温度控制为例）
**补充：二阶系统**

## 1. 温控系统建模

### 大惯性温度系统

一般而言，以温度为被控量的系统，例如电炉、热水壶等，通常可以认为是一个具有较大惯性的一阶系统，用传递函数表示为
$$G(s):=\dfrac{Y(s)}{U(s)}=\dfrac{K}{Ts+1},$$其中，$T$是惯性时间常数；$Y(s),U(s)$分别是输出信号（温度）$y(t)$和输入信号（例如加热功率）$u(t)$的拉普拉斯变换；$K$是系统静态增益，表示稳定状态下输入与输入之间的线性关系。

### 加热器的延迟
加热器的功率作用到系统的过程中存在一定的时间延迟，该延迟过程近似用一阶环节表示：
$$G_d(s)=\dfrac{1}{\tau s+1}.$$其中，$\tau$是延迟时间常数。

系统惯性与系统延迟串联作用，因此整个开环被控对象的传递函数为
$$G_o(s)=G(s)G_d(s)=\dfrac{K}{(Ts+1)(\tau s+1)},$$是一个二阶系统。
在一些对建模要求不高的场景中，可忽略延迟，即认为$\tau << T$，此时$G_o(s) \approx G(s)$。
## 2. PID控制器原理
PID是应用最广泛的一种控制算法，包含三个部分：比例P（传递函数为%K_p%）、积分I（传递函数为$\dfrac{K_i}{s}$）、微分D（传递函数为$K_d s$）。三者线性相加，组合成标准的PID控制器：
$$C(s)=K_p+\dfrac{K_i}{s}+K_ds=\dfrac{K_d s^2+K_ps+K_i}{s},$$其中，$K_p,K_i,K_d$分别称作比例、积分、微分系数。

对于一般的使用，可基于实验人工地对三个参数进行试凑和调整，以达到较好的控制效果，常用方法有飞升曲线法、Ziegler-Nichols法等。

然而，这些调参方法往往过多地依赖经验与实验，难以从理论上保证控制效果的最优；此外，在系统本身或实验环境发生变化时，先前整定完毕的参数可能不再适用，需重新整定。因此，标准PID算法存在较大的局限和不便。（此处还未考虑积分饱和与微分噪声等问题，如果考虑这些因素，标准PID的性能还将进一步下降）

为此，使用基于极点配置的PID控制器，为系统的控制性能提供理论上的支撑。

采用典型的单位负反馈控制环路，记给定参考信号及其Laplace变换分别为$r(t),R(s)$，则在控制器$C(s)$的作用下，闭环系统的传递函数为

$$\Phi(s):=\dfrac{Y(s)}{R(s)}=\dfrac{G_0(s)}{1+G_0(s)C(s)}=\dfrac{N(s)}{D(s)},$$其中，$N(s),D(s)$分别表示分子分母多项式。计算得
$$N(s)=K(K_ds^2+K_ps+K_i),\\D(s)=T\tau s^3+(T+\tau+KK_d)s^2+ (KK_p+1)s+KK_i.$$

由Laplace变换的终值定理，当参考值为阶跃信号$R(s)=\dfrac{R_0}{s}$时，有
$$y(\infty)=\lim_{s\to 0}sR(s)\dfrac{N(s)}{D(s)}=R_0.$$该结果表明，采用PID控制器，只要系统能够稳定且$K_i\neq 0$，则必定对阶跃信号无静差。

若要进一步改善暂态控制性能，应当对闭环传递函数的极点（即特征式$D(s)=0$的根）进行合理的配置。
假设期望的极点位置为$(p_1^*,p_2^*,p_3^*)$，则期望的闭环特征式为

$$\begin{aligned}D^*(s)&=T\tau(s-p_1^*)(s-p_2^*)(s-p_3^*)\\&=T\tau( s^3 +\lambda_2 s^2+\lambda_1s+\lambda_0),\end{aligned}$$其中$\lambda_0 = -p_1^*p_2^*p_3^*,\;\lambda_1 = p_1^*p_2^*+p_1^*p_3^*+p_2^*p_3^*,\;\lambda_2 = -(p_1^*+p_2^*+p_3^*).$
由此解得此时的PID参数应为：
$$\begin{aligned}K_p^*&=\dfrac{T\tau\lambda_1-1}{K},\;\;K_i^*=\dfrac{T\tau\lambda_0}{K},\;\;K_d^*=\dfrac{T\tau\lambda_2-T-\tau}{K}.\end{aligned}$$
